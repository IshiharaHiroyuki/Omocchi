## 3. インタラクション設計 🔄

### 3.1 フィードバックフロー

```mermaid
sequenceDiagram
    participant User
    participant Feedback
    participant System
    participant Admin

    User->>Feedback: 評価選択（👍/👎）
    Feedback->>System: 評価記録
    System-->>User: フィードバック受付確認
    
    opt 詳細フィードバック
        User->>Feedback: 詳細コメント入力
        Feedback->>System: コメント保存
        System->>Admin: 通知
        System-->>User: 感謝メッセージ
    end
```

### 3.2 関連FAQ表示ロジック

```mermaid
graph TD
    A[現在のFAQ] --> B{関連度計算}
    B --> C[キーワード一致]
    B --> D[カテゴリ一致]
    B --> E[タグ一致]
    B --> F[ユーザー行動ベース]
    
    C --> G[関連度スコア算出]
    D --> G
    E --> G
    F --> G
    
    G --> H[上位3件表示]
    H --> I[もっと見るボタン]
```
