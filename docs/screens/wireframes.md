## 4. インタラクションパターン 🔄

### 4.1 入力フォーム

```mermaid
sequenceDiagram
    participant User
    participant Form
    participant Validation
    participant Feedback

    User->>Form: フォーカス
    Form->>Feedback: 入力ガイド表示
    User->>Form: 入力
    Form->>Validation: リアルタイムチェック
    Validation->>Feedback: バリデーション結果表示
    User->>Form: 送信
    Form->>Feedback: 完了/エラー表示
```

### 4.2 検索インタラクション

```mermaid
sequenceDiagram
    participant User
    participant SearchBar
    participant Suggestions
    participant Results

    User->>SearchBar: キーワード入力
    SearchBar->>Suggestions: サジェスト表示
    User->>SearchBar: 検索実行
    SearchBar->>Results: インクリメンタル検索
    Results->>User: 結果表示/更新
```
